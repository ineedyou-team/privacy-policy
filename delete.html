아니쥬 계정 및 데이터 삭제 안내

아니쥬(ineedyou)는 Google 로그인을 기반으로 계정이 생성되며, 사용자가 요청할 경우 다음과 같은 개인정보 항목을 삭제할 수 있습니다.

📌 삭제 대상

  • Google 로그인 계정 정보
  • 사용자 프로필 정보 (닉네임, 생년월일, 게임 스타일, 활동 시간대 등)
  • Firebase Firestore 및 Storage에 저장된 개인 데이터

📝 삭제 요청 방법

사용자는 아래 이메일로 계정 삭제 요청을 할 수 있습니다.

  • 📧 이메일: ineedyou.team@gmail.com
  • 제목: [계정 삭제 요청] 사용자 닉네임 또는 이메일
  • 본문: 삭제 요청 사유 (선택)

요청이 확인되면 3일 이내에 모든 데이터를 삭제하고, 처리 완료 여부를 회신드립니다.

🕒 보관 및 삭제 처리

삭제 요청이 접수되면 해당 계정 및 관련 데이터는 즉시 삭제되며, 별도의 보관 기간 없이 완전히 제거됩니다.
